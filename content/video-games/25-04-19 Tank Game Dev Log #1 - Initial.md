# Shooty Thing

I've been playing this game called **"Cursed Tank Simulator** ([link](https://www.roblox.com/games/6608498361)). It's basically War Thunder but you can customize your tank the way you want; put a hull of an M4 Sherman with a Tiger II's turret, and put a TOW ATGM launcher on it. Go wild! 

It's also pretty realistic in terms of tank gun physics. There are richochets, timed fuse, overpressure, sphrapnel, and many more.

More over, this game is a passion project made during the dev's free time. The game is so fun, plus no pay-2-win aspect at all.

Though, this game has been in development for over 5 years now. I must have gone through a lot before becoming what it is today.

**Anyway.** I also really like tank combat. So I wanted to **explore how a tank game can be made**. Thus, this project is created.
